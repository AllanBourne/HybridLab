# Exchange Hybrid Lab

<p>Exchange 2013 lab designied for hybrid deployment with Office 356</p>

<ul>
<li>Azure Resource Group
<ul>
<li>VNet, Subnet, Public IP</li>

</ul>
</li>
<li>Domain Controller</li>
<li>Exchange 2013</li>
</ul>
